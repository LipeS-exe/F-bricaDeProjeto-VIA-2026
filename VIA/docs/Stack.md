# Lynx

**Tipo:** Framework multiplataforma  
**Nível:** Intermediário  
**Mobile Nativo:** Sim (renderização nativa real)  
**Performance:** Alta  

## Diferencial
- Arquitetura moderna
- Multithread
- Integração com React

## Pontos Fracos
- Comunidade pequena
- Mercado experimental

# .NET MAUI

**Tipo:** Framework multiplataforma moderno  
**Nível:** Intermediário  
**Mobile Nativo:** Sim  
**Performance:** Alta

- Mantido pela Microsoft
- Integração total com ecossistema .NET

- Código único para Android, iOS, Windows e macOS
- Ideal para quem usa C#

# Melhor Stack para Fintech

- App Mobile → Flutter ou React Native  
- App Nativo puro → Kotlin (Android) + Swift (iOS)  
- Backend → .NET ou Node  
- Banco → PostgreSQL  
- Cloud → AWS / Azure
